Quick Starter Guide

  1. 1. Start from a template

    Fork an existing sim and tweak a few words instead of starting from a blank prompt.

  2. 2. Describe structure first

    Explain the sections you want (header, main area, buttons, etc.) before going into visuals.

  3. 3. Then add behavior

    Mention interactions: “clickable cards,” “live counters,” “drag-to-adjust slider,” etc.

  4. 4. Be specific about style

    Use concrete words: “minimal, off-white background, soft shadows, single accent color.”

  5. 5. Refine with small prompts

    Iterate: “add copy buttons,” “make it mobile friendly,” “add subtle hover states.”
